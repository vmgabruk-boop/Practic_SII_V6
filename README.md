# 🚛 Truck Counter — система учета грузовиков на логистическом терминале

## Описание проекта
Веб-приложение для автоматического подсчета грузовиков на изображениях и видео с использованием предобученной нейронной сети YOLOv8.

## Технологии
- FastAPI (бэкенд)
- YOLOv8 (детекция объектов)
- HTML/CSS/JavaScript (фронтенд)
- Pandas + FPDF (генерация отчетов)

## Установка и запуск
```bash
pip install -r requirements.txt
python app.py
# Открыть http://localhost:8000
